# Rest Provider

Traefik can be configured:

  • using a RESTful api.

# Configuration

# Enable REST Provider.
[rest]
  # Name of the related entry point
  #
  # Optional
  # Default: "traefik"
  #
  entryPoint = "traefik"

# API

Path Method Description
/api/providers/web PUT update provider
/api/providers/rest PUT update provider

!!! warning For compatibility reason, when you activate the rest provider, you can use web or rest as provider value.

curl -XPUT @file "http://localhost:8080/api/providers/rest"

with @file:

{
    "frontends": {
      "frontend2": {
        "routes": {

            "rule": "Path:/test"

        },

            "rule": "Host:test.localhost"

    },
    "backends": {
      "backend2": {
        "loadBalancer": {
          "method": "drr"
        },
        "servers": {
          "server2": {
            "weight": 2,

        }
      }
    }
